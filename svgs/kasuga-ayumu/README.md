Ayumu Kasuga (春日 歩) from [Azumanga Daioh](https://en.wikipedia.org/wiki/Azumanga_Daioh), commonly known as Osaka (大阪, Ōsaka). This is a manual redraw from very last frame of the "Azumanga Daioh: The Very Very Short Movie" with some changes (softer shadows, use of gradients).

SVG contains over two dozen (not very well organized) layers, originally done in Inkscape 0.46, somewhere in 2008.

I may improve contours at some point and flush out gradients and layers.
