# My vector art collection

## What is it?

Collection of various vector art I've made over the years. Some were done for practical purposes like printing on t-shirts, some for aesthetic purposes, some for fun.

## How to use it?

Every image is available in source form (an SVG file) as well as various resolutions of raster form (a PNG file). Select one you want and download it.

Note that some images deliberately have a transparent background, so you can use them in any color you want. This is especially useful for printing on t-shirts.
